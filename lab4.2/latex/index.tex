\chapter{Шифрование методом табличной маршрутной перестановки}
\hypertarget{index}{}\label{index}\index{Шифрование методом табличной маршрутной перестановки@{Шифрование методом табличной маршрутной перестановки}}
\hypertarget{index_intro}{}\doxysection{\texorpdfstring{Введение}{Введение}}\label{index_intro}
Программа реализует шифрование и расшифровку сообщений на английском языке при помощи метода табличной маршрутной перестановки. Алгоритм разбивает текст на строки с длиной, равной ключу, и шифрует текст по столбцам.\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Использование}{Использование}}\label{index_usage}

\begin{DoxyEnumerate}
\item Пользователь вводит целочисленный ключ.
\item Выбирает операцию\+: зашифровка или расшифровка.
\item Вводит текст для обработки. Программа проверяет корректность введенных данных и выполняет выбранную операцию.
