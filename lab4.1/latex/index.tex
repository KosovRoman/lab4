Программа реализует шифрование и расшифровку русскоязычных сообщений с использованием метода Гронсвельда. Алгоритм основан на модифицированном шифре Альфа и поддерживает только заглавные буквы русского алфавита, включая символ \textquotesingle{}Ё\textquotesingle{}.\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Использование}{Использование}}\label{index_usage}

\begin{DoxyEnumerate}
\item Пользователь вводит ключ для шифрования.
\item Выбирает операцию\+: зашифровать или расшифровать текст.
\item Вводит текст для обработки. Программа проверяет корректность текста и выполняет выбранную операцию.
